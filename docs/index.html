Canary - Docs

This is where all documentation for Canary products are hosted.

CanaryEngine

CanaryEngine is an upcoming game framework for users who like to keep it simple. If you prefer to not use any external IDE's such as VSCode, or Notepad++ then this framework is for you.

With barely any setup required, you can start using cEngine in your game in just minutes!


CanaryUI

CanaryUI is a styling framework for the web that is meant for simple, and modern website UI. A Roblox compatible version is coming soon as well.


DataService v2

DataService v2 is a wrapper for ProfileService developed by Canary. It enhances what's already going on a gives extra features such as simplified GlobalUpdates.

Console

Console is a wrapper for Roblox's well known functions such as print(), or error(). This module makes debugging much easier in the long run, and it includes extra features such as timers, counters, and even table -> string conversion.

HintService

HintService is a module that replaces the legacy hints with modern, animated, and way more customizable ones.
